# contacts

Online contacts based on WEB

## Requirements
  - Redis
  - PHP (>=5.4)
  - Mysql (5.x)

## Setup
  - Install requirements
  - Rename `config-sample.inc.php` to `config.inc.php`
  - Customize `config.inc.php`
  - Run `install.php` & __Remove__ `install.php`

## Features
  - Fast & Easy to configure
  - Organized by groups
